Add a Google +1 button to the Jetpack Sharing module

== Description ==

Extends the Jetpack plugin and allows you to add a Google +1 button to the list of sharing services available under Settings > Sharing in your dashboard.

Important: for this plugin to work, you must activate [Jetpack](http://wordpress.org/extend/plugins/jetpack/) first, and activate the Sharing module.

For contact and questions, feel free to use Support or Developer contact web page at http://tusuperagente.com/contactar/

This plugin is a work in progress. Based on http://wordpress.org/plugins/scoopit-for-jetpack/ from Jeremy Herve.


== Installation ==

1. Install the Jetpack plugin, connect the plugin to WordPress.com
2. Activate the Sharing module
3. Install the Google +1 button for Jetpack plugin via the WordPress.org plugin repository, or via your dashboard
4. Activate the plugin
5. Go to Settings > Sharing, and drag the Google+1 button to the sharing area.
6. Enjoy! :)
